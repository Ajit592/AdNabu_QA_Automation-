# AdNabu QA Automation Assignment

## 📌 Overview
This project automates a key user flow of an e-commerce application:
Search for a product and add it to the cart.

## 🛠 Tech Stack
- Python
- Selenium WebDriver
- Pytest
- Page Object Model (POM)
